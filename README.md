# 📦 app-gudang

Ini adalah proyek aplikasi **Manajemen Gudang** berbasis [React Native](https://reactnative.dev) yang dibuat dengan menggunakan `@react-native-community/cli`. Aplikasi ini dirancang untuk mempermudah pengelolaan data inventaris barang secara digital, efektif, dan terstruktur.

## 🚀 Fitur Utama Aplikasi
* **Manajemen Stok:** Pencatatan barang masuk dan keluar secara real-time.
* **Manajemen Pengguna:** Pembagian akses admin dan petugas gudang.
* **Pelaporan Inventaris:** Rekap data barang untuk menghindari selisih stok.

---

## 🛠️ Teknologi yang Digunakan
* **Framework:** React Native (TypeScript)
* **Build System:** Gradle & Kotlin
* **Environment:** Expo Modules
